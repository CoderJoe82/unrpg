# Core Game Systems

## State Management
- Handles game screens and transitions
- Manages game state and save data
- Controls flow between different game modes

## Event System
- Decoupled communication between systems
- Event-driven architecture
- Clear event contracts and handlers

## Resource Management
- Handle game assets and data files
- Efficient loading and unloading
- Cache management

## Game Loop
1. Exploration
2. Encounter (Combat/Dialogue)
3. Resolution
4. Growth
5. Repeat

## Components
- Stats and attributes
- Inventory system
- Combat mechanics
- Dialogue system
- Save/Load functionality

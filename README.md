# UnRPG

A 2D Text-Driven / Menu-Based RPG Engine built with Python and Pygame CE.

**Current Version:** 0.0.1 (Pre-Alpha)
**Status:** Active Development

## Project Overview
UnRPG is a project focused on building a robust, event-driven RPG engine. It emphasizes clean architecture, separation of concerns, and a professional code structure.

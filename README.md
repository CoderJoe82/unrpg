# unrpg

Hello! This is just my personal side project. I'm doing it to help me learn python, and also consider game design concepts that I like and how to implement them over time. It will be a long, bigger project than I've ever worked on when it's done. I'm hoping it ends up a great learning experience as well as just have some fun.

To make this perfectly clear, I AM using a.i. to help me with this. I want to make this fast, but I want to also understand what I'm looking at and what I'm having the a.i. help me create. And I am testing this myself as I go along, and why nothing's being created as a full game in one go. I've got a background with MERN stack and javascript, having completed a boot camp in the past and know about coding at a decent intermediate level, I just am not familiar with python, so am just giving this a go.
